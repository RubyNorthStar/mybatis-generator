Overview

参考文档：http://blog.csdn.net/wyc_cs/article/details/9023117

执行方式：
1、进入mybatis-generator-core-1.3.2.jar 所在的目录
2、配置generatorConfig.xml文件内容
3、执行脚本：java -jar mybatis-generator-core-1.3.2.jar -configfile generatorConfig.xml -overwrite
